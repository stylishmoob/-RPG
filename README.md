# 人生RPG

## 概要

人生をRPGのように成長させることを目的とした習慣化支援Webアプリです。

日々の活動時間を記録し、その行動に応じてキャラクターのステータスが成長します。

「現実の行動がゲームの経験値になる」というコンセプトで開発しています。

---

## 主な機能

### 活動記録

* ストップウォッチによる活動時間の計測
* カテゴリーごとの活動管理
* 活動履歴の保存

### ステータスシステム

* 活動内容に応じてステータスが上昇
* 筋力（STR）
* 知力（INT）
* 技術力（SKL）
* 創造力（CRE）
* 意志力（WIL）

### 活動分析

* カテゴリー別活動時間集計
* 日別活動時間集計
* グラフによる可視化

## 開発目的

毎日の行動を数値化し、

* 自分が何に時間を使っているか
* どの能力が成長しているか

を可視化するために開発しました。
